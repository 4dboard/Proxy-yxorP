# Async

Async Helper class.

Use only if you know what you're doing!!!

Usage:

Cockpit::instance()->helper('async')->exec('
   sleep(10);
   file_put_contents(COCKPIT_DIR."/debug.txt", $test);
', ['test' => 222]);

* Full name: `\App\Helper\Async`
